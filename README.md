Verificação de endereço pelo CEP.

Projeto para autocompletar um endereço com a utilização de uma API (VIACEP) de verificação de CEP, 
recebendo os dados no formato json e utilizando para preenceher alguns campos.

Desenvolvido como parte do curso do Matheus Battist - Hora de Codar.
